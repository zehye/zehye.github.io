## python

#### 리터럴

변하지 않는 고정된 데이터 자체의 표현

* 5 (정수형 데이터)
* "Fastcampus" (문자열 데이터)
* 1.2543 (부동 소수점 데이터)

#### 표현식 (expression)

값을 의미하는 표현 또는 값을 반환하는 표현

#### 구문 (statement)

값의 의미를 지니지않으며, 어떠한 목적을 수행하는 코드

```
for char in '안녕하세요':
  print(char)
```
